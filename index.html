How to use?

There are three ways:

1. Enter youtube URL

Just enter URL to the form above!

2.Add script to the bookmark bar

Add this script to the bookmar bar of your favourite browser:

javascript:void(location.href='https://purify-video.github.io/?v='+location.href)

Then click this bookmark on any youtube page.

3.Use user script

Use this userscript with Tampermonkey plugin

FAQ

What's this?

This app removes distracting comments and sidebar related videos.

It can be used by teachers, professors, students, parents, organizers and others for:

  • Clean and minimalistic interface to view YouTube videos
  • Removing inappropriate ads and comments from videos
  • Removing unwanted related videos at the end to show to students
